# FormAI - SIMPLE VERSION

## How to start (SUPER SIMPLE):

1. **Open terminal** in this folder
2. **Run:** `cargo run`
3. **Open browser:** http://localhost:5511
4. **Done!**

## Navigation:
- **Dashboard:** http://localhost:5511
- **Profiles:** http://localhost:5511/profiles
- **Automation:** http://localhost:5511/automation

## That's it!
- No complex routing
- No background processes fighting
- Just simple HTML navigation
- Works every time
